# Challenge: Characters manager

Introduction project concerning API.

## Objectives of this project:

- Write semantic HTML.
- Use SASS + B.E.M. Methodology
- JavaScript API's
  - API - GET, POST, PUT, DELETE
  - Axios
  - Async / await
- JavaScript modern anatomy
  - Refactor code
  - Modules
  - Packet Manager
  - Linters
- Create a Readme.
